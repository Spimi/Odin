What do HTML and CSS stand for?
HTML = Hypertext Markup Language (Structure) CSS = Cascading Style Sheets ( colors, font styles, layout, responsive features etc.)

Between HTML and CSS, which would you use for putting paragraphs of text on a webpage?
HTML

Between HTML and CSS, which would you use for changing the font and background color of a button?
CSS

What is the difference between HTML, CSS and JavaScript?
We use HTML for the structure of the website, CSS for the style, and Javascript for interactive elements.

What is an HTML tag?
HTML tags wrap up pieces of content into HTML elements. The opening and closing tags tell the browser what content the element contains.

What are the three parts of an HTML element?
Opening tag, content, and a closing tag. : <p>SomeText</p>

-----------------------------------------------------------------

What is the purpose of the doctype declaration?
Doctype declaration tells the browser what version of HTML it should use to render the document.

What is the HTML element?
HTML element is the root element, every other element a descendant of it.

What is the purpose of the head element?
We put meta-informations about our webpage and stuff required for our webpages to render correctly in the browser.

What is the purpose of the body element?
We put every content that will be displayed to the users in the body tag, text, images, lists, links, etc.

How do you create a paragraph in HTML?
With the paragraph element. <p></p>

How do you create a heading in HTML?
With the heading elements, from 1 to 6. <h1></h1> - <h6></h6> The h1 is the most important, the h6 is the lowest level.

How many different levels of headings are there and what is the difference between them?
1-6. Heading give hierarchy to the page. h1 is the most important, and should used for the heading of the overall page. Displayed larger and bolder than the other texts.

What element should you use to make text bold and important?
<strong></strong> It affects tools like screen readers.

What element should you use to make text italicized to add emphasis to it?
<em></em> Places emphasis on the text, which again afftect things like screen readers.

What relationship does an element have with any nested elements within it?
A parent - child relationship. The nested elements are the children, they are nested within is the parent.

What relationship do two elements have if they are at the same level of nesting?
Siblings
