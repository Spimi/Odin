Did you use all three methods of adding CSS to an HTML file?
Yes I use external, internal, and inline method.

Did you properly link the external CSS file in the HTML file?
Yes, I use <link rel="stylesheet" href="style.css">.

Does the div element have CSS added via the external method?
Yes, the div element CSS properties defined in style.css

Does the p element have CSS added via the internal method?
Yes the p element CSS properties defined inside the html head element, with <style> tags.

Does the button element have CSS added via the inline method?
Yes the button element CSS properties defined inside the button opening tag. <button style="">
